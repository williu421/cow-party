Cow Party is a simple version of Mario Party that runs in Python.
Players take turns moving around a board. The squares they land on trigger
different events, and at the end of each turn the players play a minigame together. 
The objective is to collect beans, which is done by landing on blue squares and doing well 
on the minigames.

requirements: 
    pygame 
    socket
    everything else should come with normal python installation

How to play: 
1) In the constants file, change the IP address to the IP address of the server player (if playing two player)
2) start the server file  (if playing two player)
3) Run the main.py files

Fonts you need to download: : 
    Broken Home
    28 Days Later 
    Gravedigger
